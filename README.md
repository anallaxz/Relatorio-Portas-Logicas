# Relatório Técnico: Portas Lógicas Digitais

Este repositório contém o relatório técnico e os detalhes das simulações de portas lógicas realizadas como parte dos estudos de Sistemas Digitais e Microprocessadores.

## 📋 Resumo do Projeto
O projeto apresenta o desenvolvimento de uma atividade prática envolvendo portas lógicas digitais utilizando circuitos integrados da família 74HC. O objetivo principal foi compreender o funcionamento das portas AND, OR, NOT, NAND e NOR por meio de análise prática e simulação.

## 🛠️ Materiais e Ferramentas
**Software de Simulação**: Tinkercad 
* **Circuitos Integrados (Família 74HC)**:
    * 74HC08 (AND) 
    * 74HC32 (OR) 
    * 74HC04 (NOT)
    * 74HC00 (NAND) 
    * 74HC02 (NOR)
      
**Componentes**: Protoboard, LEDs, resistores (1 kΩ), chaves, fonte de 5V e jumpers

## 🚀 Metodologia e Resultados
Os circuitos foram montados em protoboard virtual no ambiente Tinkercad. Para cada porta lógica:
1. Foi desenvolvido o esquema elétrico baseado no datasheet.
2. Realizou-se a montagem prática com alimentação de 5V.
3. Foram aplicadas combinações de entradas para validar as **Tabelas-Verdade**.

As medições confirmaram níveis de tensão próximos a 5V para nível lógico alto (1) e 0V para nível lógico baixo (0), validando o comportamento teórico esperado.

## 📚 Conclusão
A atividade consolidou conhecimentos sobre a base dos sistemas digitais e desenvolveu habilidades essenciais de montagem e análise de circuitos eletrônicos.
